## Задачи на проект
+ Рефакторинг репозитория. Привести в порядок. 
+ Выработка системных требований.

+ Кроссплатформенность 
+ Нагрузочное тестирование

+ Обеспечить работоспособность на компе, на локалке и при выделенном домене.
